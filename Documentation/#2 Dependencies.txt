	Function(s)/Classes/Methods Used From The Module : path(), makedirs(), remove(), system()
	Purpose :
		path():
			To Check If The Desired File Exists.
		makedirs():

		remove():
			To Permanently Delete An File.
		system():
			To Call In The System Command(s).
				"cls" - To Clear The Console.
